Demonstrações ao Vivo por Integrante

Abra o sistema, mostre a sua parte funcionando e use os prints do código como apoio
Para que serve este conjunto: diferente dos guias de apresentação (em que você explica o código passo a passo), aqui o foco é demonstrar o sistema funcionando. Cada integrante abre a aplicação, navega até a sua parte, mostra o recurso em ação e narra o comportamento — sem precisar explicar o código linha a linha.
🖥️1. Deixe o sistema rodando (npm run dev → localhost:5173).
🧭2. Vá até a aba da sua parte e demonstre o que cada recurso faz.
📄3. Os prints do código ficam na própria página — mostre cada um e leia só a legenda "o que faz".
